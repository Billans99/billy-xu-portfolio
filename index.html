Projects

Pokidex Paradise

Pokedex Paradise is the first project I have built without a tutorial and is a single page React web app that fetches data from PokeAPI and displays pokemon information in ascending order of ID. Simply hover the cursor above a pokemon's name to display its information. The idea to build this app was inspired by an Ania Kubow YouTube video/short detailing PokeAPI as one of the most used APIs ever. This was quite interesting and inspired me to build this app.

The sort of information users can find include:

  • Appearance
  • Name
  • Type (and corresponding colour of type)
  • Abilities
  • Base stats

Anime App

This Anime App is the second project I have built without a tutorial. This React web app fetches data from the free API, Jikan. I wanted to create a movies app without doing the same thing as everyone else, so I decided to create a similar app based on anime. Users are able to find various animes as well as indepth information by clicking the "view" button. Users can also search for particular anime using the search function.

I used some React Bootstrap components to create the buttons, modal, as well as some others elements.

The sort of information users can find include:

  • Summary
  • Background
  • General info
  • Reviews
  • Characters
  • News
  • Stats
  • Pictures
  • Recommendations
  • Theme music
  • Relations
  • External links
  • Streaming platforms

One of my biggest takeaways of the project is that some free APIs can be unreliable. Unfortunately, some of the data won't load due to the API being free. The console will ocassionally show error (429) "Too Many Requests", however, it does all work. If something is not loading, you can try refreshing the page.
