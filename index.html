Projects

Vital Help

Vital Help is a multi-page, full-stack node app built on React. The frontend uses Material UI and React Bootstrap, while the backend utilises Node and Express for the backend. MongoDB and Mongoose is used for the database. The application fetches mental health statistical data from my own RESTful API and MongoDB database. Additionally, Patch, Post, and Delete requests can be used to update the data. The app is designed to raise awareness for men's mental health by revealing statistical data and providing external support links for mental health.

For routing, I used React Router to navigate between pages which includes:

  • Home
  • Statistics
  • About
  • General Support
  • Crisis Support

123Moovies (movies app)

123Moovies is a movies web application that fetches data from the TMDB API. The application displays new movies along with their respective ratings. These ratings are color coded to indicate the quality of the movie based on reviews. Hovering over each movie will transition a brief summary of the movie. Users are also able to search for specific movies by typing in the search bar. HTML, CSS, and Javascript was used to create this application.

Weather App

This weather app fetches data from the OpenWeatherMap API. Users can search for a city and the app will display the current weather. The app displays the temperature, humidity, and wind speed. The background of the app changes depending on the weather of the city. HTML, CSS, and Javascript was used to create this application.

Simply type in the city you would like to search for in the search bar and click the search button.

Pokidex Paradise

Pokedex Paradise is a single page React web application that fetches data from PokeAPI and displays pokemon information in ascending order of ID. Simply hover the cursor above a pokemon's name to display its information.

The sort of information users can find include:

  • Appearance
  • Name
  • Type (and corresponding colour of type)
  • Abilities
  • Base stats
